CSS grid:

CSS grid is a two-dimensional layout method thatâ€™s gonna handle both rows and columns. The hierarchy is as same as flex-container. You can refer complete CSS guid for CSS grid.

All of we need to do is applying display:grid property to the block elements and defining number of rows and columns by using grid-template-coloumn/row properties.The main motive of introducing grid module is to design a complex layouts that may not be possible with flex-box.

Areas of application:

  • When you have a complex designs and maintainable websites grid layout is prefered over flex.
  • When your layout consists of gap between two block elements, take grid concept.
  • When you need to overlap elements grid has an excellent feature with grid coloum and row property.
  • When your website design should be planned for layout-first method, then grid is the better option.

Drabacks:

  • Not supported in every browser.
  • Choosing grid is a bit difficult for resposive designs.
